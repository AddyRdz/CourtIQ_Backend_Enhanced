# CourtIQ
* This is the backend for the FullStack App named CourtIQ. CourtIQ is built with express and mongoose. This database allows you to add your own comments and ratings to players and teams. 

## Routes (Players)
* /players GET
* /players:id GET
* /players:id PATCH
* /players POST
* /players:id DELETE

## Routes (Teams)
* /teams GET
* /teams:id GET
* /teams:id PATCH
* /teams POST
* /teams:id DELETE
